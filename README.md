## About

This repository is my long-term Infrastructure knowledge base.

The goal is not to collect notes, but to build an engineering-oriented knowledge system through continuous learning, experiments, troubleshooting, and documentation.

My long-term roadmap:

```text
Linux
    ↓
Computer Network
    ↓

    ↓
Docker
    ↓
Kubernetes
    ↓
Distributed Systems
    ↓
Cloud Native
    ↓
AI Infrastructure / LLMOps
```

This repository follows an engineering-first learning methodology.

- Learn by building
- Learn by troubleshooting
